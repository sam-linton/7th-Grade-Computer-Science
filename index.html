First HTML

Learn the basics of creating a Hyptertext Markup Language (HTML) page.

Notes

Web programs consist of:

  • HTML - markup language which defines the content and structure
  • CSS - styling language which controls the style and layout
  • Javascript - programming language that allows interactivity and data manipulation.

HTML documents are made of elements, each with an opening and closing tag. For example, here is a paragraph element: <p> This is a paragraph </p>

HTML documents always have one element inside them: the html element. The html element contains two other elements: the head and the body. The head element contains information about the document, such as the title. The body element contains the content of the document. Here is the basic structure of an HTML document: 

                  
                    <html>
                      <head>
                        <title> My First Web Page </title>
                      </head>
                      <body>
                        <p> This is a paragraph </p>
                      </body>
                    </html>

HTML elements may have attributes that define properties. Attributes must be assigned in the opening tag of an element.

HTML elements may have contents. Contents appear between the opening and closing tags of elements.

Here are examples of some common tags:

Tag Description Example Notes
h1, h2, h3, etc headings <h1> This is a header </h1>
p paragraph <p> This is a paragraph </p>
img image <img src="hamster.jpg" alt="Image of a hamster"> Note there is no close tag required!
a anchor (link) <h1> This is a header </h1> An image (or other elements) can be used instead of text inside the anchor tag

Styling

Learn how to add styling to an HTML page using Cascading Stylesheets (CSS).

Notes

Cascading Style Sheets (CSS) allow programmers to control the style of web programs, overriding the browser default behavior.

Add CSS styles with a selector (what to style) and a ruleset (how to style it). 

    
      p {
        color: red;
      }

Selectors can select elements, ids, or classes:

Selector Type HTML example CSS Selector
Element
Class <p class=”important”></p> .important
Id <p id=”first”></p> #first

Best practice to add CSS is to use a separate file (such as style.css) and link the html to it in the header: 
  
    <link href=”style.css” rel=”stylesheet type = “text/css”>

The Box Model determines the spacing around elements:

There are many units for spacing in CSS. Here are three of the most common:

  • pixels - based on the size of a monitor’s pixel size
  • rems - relative to the size of an “m”
  • percentage - % of the size of whatever contains the element
I recommend that you avoid using pixels!

Hint: to “turn off” the browser’s default settings, consider having this as the first thing in your CSS file: 

  
    *  {
      margin: 0rem;
      padding: 0rem;
      box-sizing: border-box;
    }
